# 🐧 Linux Fundamentals Part 1 - TryHackMe

## 📚 O que aprendi

Neste módulo aprendi os primeiros comandos básicos do Linux:

- echo → mostra texto no terminal
- whoami → mostra o usuário atual
- ls → lista arquivos e pastas
- cd → entra em pastas
- pwd → mostra onde estou
- cat → mostra conteúdo de arquivos
- find → procura arquivos
- grep → busca dentro de arquivos

---

## 💻 Comandos importantes

```bash id="cmd1"
echo "texto"        # mostra texto no terminal
whoami              # mostra usuário atual
ls                  # lista arquivos
cd pasta            # entra em pastas
pwd                 # mostra caminho atual
cat arquivo.txt     # mostra conteúdo do arquivo
find -name *.txt    # procura arquivos
grep "THM" file     # busca texto dentro do arquivo
